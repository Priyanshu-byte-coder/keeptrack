KeepTrack

You download. We remember.

Every file you download makes sense in the moment. Two weeks later, your Downloads folder is a mess. KeepTrack captures that decision at download time.

Download v1.2.0 Star on GitHub

Powerful Features

Everything you need to keep your downloads folder clean and organized, natively integrated into Chrome.

Smart Classification

Automatically classifies every download as Keep, Temporary, or Ambiguous using intelligent heuristics based on file type, filename keywords, and source domain.

Gentle Notifications

Only interrupts for ambiguous downloads. High-confidence decisions are entirely silent.

Auto Expiration

Delete individual files or bulk-delete expired files directly from the popup. One click, files gone from disk.

Privacy-First & Local

Zero telemetry. Zero network calls. All data stays in your browser's local storage. Fully open source — read every line of code.

Install KeepTrack

Choose your preferred installation method below.

Download the latest release

Download the packaged ZIP, extract it, and load it directly into Chrome.

Download ZIP
  1. Extract the downloaded ZIP
  2. Open chrome://extensions
  3. Enable Developer Mode (top right)
  4. Click Load unpacked and select the extracted folder

Install via Terminal

One command. Auto-detects and installs the latest version. Requires curl and unzip.

curl -fsSL https://raw.githubusercontent.com/Priyanshu-byte-coder/keeptrack/gh-pages/install.sh | bash

Install via PowerShell

Run the following command in PowerShell as Administrator.

irm https://raw.githubusercontent.com/Priyanshu-byte-coder/keeptrack/gh-pages/install.ps1 | iex

Clone from GitHub

For developers who want to inspect or modify the code.

git clone https://github.com/Priyanshu-byte-coder/keeptrack.git

Then load unpacked from chrome://extensions.

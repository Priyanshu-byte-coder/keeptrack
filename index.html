Features

Smart Classification

Automatically classifies every download as keep, temporary, or ambiguous using file type, filename keywords, and source domain.

Gentle Notifications

Only interrupts for ambiguous downloads. High-confidence decisions are silent. One tap to keep or toss.

Delete Expired Files

Delete individual files or bulk-delete all expired files directly from the popup. One click, files gone from disk.

Label Toggle

Change your mind anytime. Toggle any file between Keep, Temporary, and Ambiguous from the Settings page with one click.

Clean Up Now

Skip the expiry wait. Instantly flag all temporary files for review when you want a clean Downloads folder right now.

Privacy-First

Zero telemetry. Zero network calls. All data stays in your browser. Fully open source — read every line of code.

Install

Recommended

Direct Download

Download the ZIP, unzip, load in Chrome.

Download ZIP
  1. Download the ZIP above
  2. Unzip to any folder
  3. Open chrome://extensions
  4. Enable Developer Mode
  5. Click Load unpacked → select folder

Terminal

One command. Auto-detects latest version.

curl -fsSL https://raw.githubusercontent.com/Priyanshu-byte-coder/keeptrack/gh-pages/install.sh | bash

Works on macOS and Linux. Requires curl and unzip.

Clone from Source

For developers who want to inspect or modify.

git clone https://github.com/Priyanshu-byte-coder/keeptrack.git

Then load unpacked from chrome://extensions.
